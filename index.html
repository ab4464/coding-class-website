Lesson 1


Overview

To get a feel for HTML and CSS, we're going to "tinker" with an existing program rather than build one from scratch one Day 1. What this means is that you'll be manipulating and thinking about a very simple and silly webpage. I will provide you with a few guiding questions to help you direct your attention and activities to the key patterns and relationships in the code, but I encourage you to formulate your own questions and ideas about how the code works and to test out your hypotheses.




Tinker Activity

  • Start by opening the program. Click the "Edit in JSFiddle" button below.
  • Then, compare what you see in the code and map the relationships between the code and output.
  • After you're vaguely familiar with the program, use the guiding prompts and questions to manipulate the actual code one thing at a time and see what happens.
  • Finally, articulate and share your understandings and findings with me by answering the questions below.
Guided Tinkering with HTML
Change the OL tags to UL tags in the HTML code. What happens when you do that?

Make a new list of the top 3 things cats love and copy-paste your html code here.

Delete the closing H3 tag in the "intro" section. What happens when you do that?
Select the answer that most accurately describes the change in output.

Guided Tinkering with CSS
Change the value of the "text-decoration" attribute for the h4, h5, and h6 elements from "underline" to "none". What happens when you do that? Select best answer from the dropdown menu.

Add another CSS styling rule to the h1, h2, h3, h4, h5, and h6 elements. Right below "font-weight: bold;" you should type "font-style: oblique;" What happens when you add this styling rule? Select all that apply by holding down the command/option key.

How would you change the color of the title, "Meow meow meow meow"? Copy-paste your CSS code here.
